# ico-website-dev
the server has implemented livereload for dev

### Requirement 

- node v9.0.0 (npm v5.5.1)
- npm install gulp-cli -g


### Run dev

- npm run startDev
- http://localhost:7000/


### Run Prod

- npm run startProd
- http://localhost:3000/



### Info

- Folder react =./client
- Folder express =./server
